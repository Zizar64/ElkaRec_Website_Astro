# Notre régie mobile multicaméras : retour d'expérience

La mise en place d'une régie mobile multicaméras a été l'un de nos projets les plus ambitieux. Retour sur cette aventure technique et humaine.

## Le contexte

Face à la demande croissante de productions multicaméras pour des événements variés (concerts, conférences, podcasts), nous avons décidé de développer notre propre solution mobile et flexible.

## Les objectifs

Notre régie devait être :
- **Transportable** : Facile à déplacer et à installer
- **Flexible** : Adaptable à différents types d'événements
- **Performante** : Qualité broadcast
- **Accessible** : Budget raisonnable et maintenance simple

## La solution technique

### Architecture globale

Nous avons opté pour une architecture hybride mêlant :
- Flux SDI pour les caméras principales
- NDI pour les sources secondaires et écrans de retour
- Réseau IP 10 Gigabit pour les interconnexions
- Enregistrement multi-formats (ISO, Programme, Backup)

### Équipements principaux

#### Vidéo
- Mélangeur vidéo avec effets temps réel
- 4-6 caméras selon les besoins
- Moniteurs multiviewer
- Convertisseurs SDI/NDI

#### Audio
- Console de mixage numérique
- Systèmes de monitoring
- Intercom intégré

#### Réseau
- Switches 10GbE
- Redondance des liens critiques
- VLAN dédiés par type de flux

## Les défis rencontrés

### Challenge 1 : La gestion des câbles

Solution : Création de looms pré-câblés et code couleur strict

### Challenge 2 : La latence

Solution : Optimisation du réseau et choix judicieux des protocoles

### Challenge 3 : La formation

Solution : Documentation détaillée et sessions de formation régulières

## Les résultats

Après plusieurs mois d'utilisation sur différents types d'événements :

- **50+ événements** couverts avec succès
- **200+ heures** de direct sans incident majeur
- **Satisfaction client** constamment élevée
- **Évolution continue** grâce aux retours d'expérience

## Cas d'usage concrets

### Podcast ELKAST

Notre régie permet de produire le podcast ELKAST avec :
- 3 caméras sur les intervenants
- Incrustation de graphiques en temps réel
- Streaming multi-plateformes
- Enregistrement pour post-production

### Festivals

Configuration adaptée pour :
- Captation de concerts
- Multi-diffusion (écrans, stream, enregistrement)
- Gestion des retours scène
- Production de highlights en temps réel

## Enseignements
